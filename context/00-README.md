# Portfolio — Documentation

Documentation structurée de ton portfolio, au format markdown. **Même style que `.context/` de PulsaBot** : numérotation, tables, sections claires, prêt à partager ou à alimenter un site.

---

## Utilisation

| Usage | Action |
|-------|--------|
| **Personnaliser** | Remplacer les `[placeholders]` par tes infos |
| **Exporter** | Générer un site (Astro, Next), un PDF, ou partager les MD |
| **Contexte IA** | Référencer `@portfolio/` pour restaurer le contexte |
| **Copier-coller** | Utiliser les extraits de `06-EXTRAS.md` (pitch, FAQ) |

---

## Fichiers

| # | Fichier | Contenu |
|---|---------|---------|
| — | `CAHIER-DES-CHARGES.md` | Spécifications, périmètre, stack, livrables |
| — | `ROADMAP.md` | Plan de développement par phases |
| 01 | `01-OVERVIEW.md` | Présentation, parcours, objectifs, points forts |
| 02 | `02-STACK.md` | Technologies, versions, usage par projet |
| 03 | `03-PROJETS.md` | Projets détaillés (PulsaBot pré-rempli) |
| 04 | `04-COMPETENCES.md` | Compétences techniques, méthodologie, langues |
| 05 | `05-CONTACT.md` | Liens, disponibilité, préférences |
| 06 | `06-EXTRAS.md` | Pitch, FAQ, réponses types pour entretiens |
